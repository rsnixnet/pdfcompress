PDF Scan Compressor
====================

Запуск готовой portable-сборки:
1) Откройте папку dist\PdfScanCompressor
2) Запустите PdfScanCompressor.exe

Сборка portable версии:
1) Установите Python 3.10+ (64-bit) и добавьте его в PATH.
2) Запустите build_portable.bat
3) Готовая папка будет в dist\PdfScanCompressor

Основные функции:
- Добавляйте PDF-файлы или папки, либо перетащите их в список.
- Выберите пресет или перейдите в Advanced для тонкой настройки.
- Укажите выходную папку (опционально) и запустите обработку.
